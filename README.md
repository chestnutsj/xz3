A download tool

```
Usage: xz3 [OPTIONS] <URL>

Arguments:
  <URL>  Sets the URL to download from

Options:
  -n, --name <NAME>              file name
  -t, --threadsize <THREADSIZE>  Sets the thread [default: 10]
  -m, --max-retry <MAX_RETRY>    Set the retry time [default: 10]
  -h, --help                     Print help
  -V, --version                  Print version
```
